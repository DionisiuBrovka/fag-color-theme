# Support

## Getting help

- **Bug reports and feature requests** — [GitHub Issues](https://github.com/DionisiuBrovka/fag-color-theme/issues)
- **Questions** — open a [GitHub Discussion](https://github.com/DionisiuBrovka/fag-color-theme/discussions)

## Something looks wrong?

If a color looks off or a UI element is hard to read, open an issue with:
- A screenshot
- The theme variant (e.g. "FAG-green dark")
- The language or UI area affected (e.g. Python syntax, terminal, sidebar)
